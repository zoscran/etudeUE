## 1. Guide d'utilisation

### Système de référencement

Chaque source est identifiée par un **code unique** composé de :
- **Catégorie** (2 lettres) : II (Institutionnel International), GF (Gouvernemental France), GE (Gouvernemental Étranger), OP (Organisation Professionnelle), PS (Publication Scientifique), DS (Données Statistiques), PA (Presse/Analyses)
- **Numéro séquentiel** (3 chiffres)

**Exemple** : `[II-001]` = Source institutionnelle internationale n°001

### Niveaux de fiabilité

| Niveau | Description | Symbole |
|--------|-------------|---------|
| ★★★★★ | Source primaire officielle (rapports gouvernementaux, décisions réglementaires) | 🔵 |
| ★★★★☆ | Organisation internationale reconnue (AIEA, OCDE, ONU) | 🟢 |
| ★★★☆☆ | Organisation professionnelle/industrielle (EPRI, WNA, GIF) | 🟡 |
| ★★☆☆☆ | Publication scientifique peer-reviewed | 🟠 |
| ★☆☆☆☆ | Presse spécialisée, analyses sectorielles | ⚪ |

### Comment citer dans les parties

Dans chaque partie de l'analyse, les affirmations sourcées sont suivies du code de référence :

> *"Les réacteurs nucléaires peuvent fonctionner 80 ans ou plus"* `[GE-003]`

Le lecteur peut alors consulter ce glossaire pour vérifier la source complète.

## 14. Méthodologie de validation

### Processus de vérification

1. **Identification** : Toute affirmation factuelle doit être attribuée à une source
2. **Classification** : La source est classée selon sa fiabilité (★ à ★★★★★)
3. **Vérification** : L'URL ou la référence doit permettre l'accès direct
4. **Citation** : Les citations exactes sont entre guillemets
5. **Mise à jour** : Les sources sont datées et vérifiées régulièrement

### Critères de rejet

Une source est rejetée si :
- Aucune attribution institutionnelle claire
- Données non vérifiables
- Conflit d'intérêt manifeste non déclaré
- Contradiction avec sources primaires sans explication

### Traitement des incertitudes

Lorsque les sources divergent ou présentent des incertitudes :
- Les différentes estimations sont présentées
- La fourchette est indiquée
- Les raisons de divergence sont expliquées
- Le niveau de confiance est précisé
